#Patina - A Rust Shell

A shell written in rust with design ideas from fish and bash

Nowhere near feature complete

##Features

- Fish-like syntax (not trying to be bash/POSIX compatible)
- Rudimentary tab completion

More to come

##Built-in Commands

- cd
- clear
- exit

More to come
